# myfeed
Add arguments to RSS feed in guile

# requirements
Guile 3.0 or higher

# How to install
download file
create .config/myfeed/ folder (mkdir .config/myfeed) 
mv config.scm .config/myfeed
edit config.scm to have the folders you want to store your files in. 

move myfeed.scm to a path folder and give chmod 755.
